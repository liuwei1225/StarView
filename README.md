### 自定义评分星星控件绘制,支持布局或者代码中自定义颜色,星星数量,半星绘制等

* 亮星颜色
app:starColor
* 暗星颜色
app:bgColor
* 暗星绘制方式
app:starStyle
  - FILL 填充
  - STROKE 描边
  - FILL_AND_STROKE 填充并描边
* 星星总数量
app:starCount
* 亮星数量
app:rating
* 绘制高度
app:starSize
* 画笔粗细
app:paintSize
* 星星之间的间距
app:starMarge
* 是否可滑动调整
app:isChange
* 是否可绘制半星
app:half
